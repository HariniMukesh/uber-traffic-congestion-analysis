## Project Overview
This project analyses Uber traffic data to identify congestion patterns and predict traffic volume during peak hours using machine learning techniques.

## Objectives
- Identify peak traffic hours
- Analyse congestion trends
- Build a predictive model for traffic volume
- Provide insights useful for urban traffic planning

## Dataset
- Source: Uber traffic data (CSV files)
- Features include time, location, traffic volume, congestion level
